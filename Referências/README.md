# Referências

Esta pasta foi criada para armazenar materiais de apoio utilizados no desenvolvimento do projeto.

Aqui poderão ser adicionados futuramente:

- Artigos e textos acadêmicos  
- Imagens e documentos históricos  
- Links úteis e sites de pesquisa

- Materiais utilizados como base para o conteúdo do aplicativo

Organizar todas as referências e fontes utilizadas no projeto, facilitando consultas futuras e a manutenção das informações apresentadas no aplicativo.
